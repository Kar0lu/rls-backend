## Creating a Virtual Environment
### Windows
```
python -m venv venv
.\venv\Scripts\activate
pip install ./backend/requirements.txt
```
### Linux/ macOS
```
python -m venv venv
source venv/bin/activate
pip install backend/requirements.txt
```
